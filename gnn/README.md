# gnn

Graph Neural Network toxicity and wake modeling component.

## Responsibilities

- Compute street-level toxicity-aware edge weights using physics + graph learning.
- Model Gaussian plume dispersion from pollution sources.
- Fuse real-time observations with historical context for robust risk estimates.
- Generate training tensors from stream-backed state for model updates.

## Main Files

- `pi_gnn.py`: Core GNN model architecture for toxicity prediction.
- `plume_physics.py`: Gaussian plume model for pollutant transport and dispersion.
- `angular_diffusion.py`: Wind-aware diffusion modeling for plume evolution.
- `edge_weights.py`: Edge-level toxicity weight computation and aggregation.
- `graph_builder.py`: OSMnx graph loading, projection, and caching.
- `wake_predictor.py`: Upwind/downwind toxicity propagation logic.

## Model Training

Trained models use spatio-temporal GNN architecture on stream-backed state.

### Datasets

- Training data generated by `ingestion/pull_gnn_training_data.py`.
- Tensors built from fused grid state and edge toxicity snapshots.
- Historical context joins stream observations with 2022-2026 data archive.

### Checkpoints

Saved weights available in `model_weights/`:

- `pi_gnn_v1.pt`: Latest trained model (PyTorch format).

## Physics Layer

The GNN layer implements **city-agnostic physics** with three components:

1. **Gaussian Plume Model** (`plume_physics.py`)
   - Models pollutant transport and dispersion from source streets.
   - **Pasquill-Gifford stability classes (A–F)** dynamically determined from weather.
   - Dispersion coefficients σy(d, stability) and σz(d, stability) scale with distance.
   - Urban canyon correction: amplifies concentration in high building-density areas (BD > 0.7).

2. **Angular Diffusion** (`angular_diffusion.py`)
   - Wind-driven diffusion perpendicular to mean flow.
   - **Canyon tunneling**: high-density streets blend wind toward street bearing (85% deflection).
   - Downwind concentration bleed computed as Gaussian in wind direction.

3. **Respiratory Minute Volume (RMV)** (`router/edge_cost.py`)
   - Mode-dependent biological intake rate (EPA standards):
     - Walking: 1.2 m³/hr
     - Cycling: 3.5 m³/hr (2.9× pedestrian)
       - Two wheeler (`two_wheeler`): 0.6 m³/hr (current motorized baseline)
       - Legacy aliases: `driving`, `car` (reference/compatibility)
   - Inhaled dose = Concentration × RMV × Travel Time (in µg)

### Configuration

Physics parameters are **centralized** in `shared/physics_config.py`:
- `StabilityClass` enum (A–F)
- `get_pasquill_stability(wind, solar_rad, is_night)` — dynamic stability lookup
- `get_stability_dispersion_params(stability)` — Pasquill coefficients
- `get_respiratory_minute_volume(mode)` — EPA RMV constants
- `UrbanCanyon`: building density thresholds, canyon deflection strength
- `CITY_INSTANCES` dict: extensible city registry for multi-city support

Physics is **universal** across cities; only data varies (weather, graph, AQ sources).

For detailed physics, see [docs/PHYSICS.md](../docs/PHYSICS.md) with full EPA sourcing.

## Integration with Routing

Edge weights computed by the GNN layer are consumed by the router's A* search to reduce exposure rather than only minimizing distance.

Example usage:

```python
from gnn.graph_builder import build_and_load_graph
from gnn.edge_weights import EdgeWeightAggregator

# Load cached UTM graph or build from OpenStreetMap
graph = build_and_load_graph()

# Compute toxicity weights given stream state
aggregator = EdgeWeightAggregator(graph)
weights = aggregator.compute_from_streams()
```
